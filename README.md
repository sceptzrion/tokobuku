## Run Locally

Clone the project

```bash
  git clone
```

Go to the project directory

```bash
  cd project-name
```

-   Copy .env.example file to .env and edit database credentials there

```bash
    composer install
```

```bash
    php artisan key:generate
```

```bash
    php artisan artisan migrate

    php artisan db:seed
```

```bash
    php artisan storage:link
```

```bash
    php artisan serve
```
